# To-Do App Backend\n\nFastAPI backend for the to-do application.\n\n## Project Structure\n\n```\nbackend/\n├── app/\n│   ├── __init__.py          # Package initialization\n│   ├── main.py              # FastAPI app entry point\n│   ├── config.py            # Configuration management\n│   ├── database.py          # Database connection & session\n│   ├── models/              # SQLAlchemy ORM models\n│   ├── routers/             # API route handlers\n│   ├── schemas/             # Pydantic request/response schemas\n│   └── utils/               # Utility functions & helpers\n├── tests/\n│   ├── __init__.py\n│   ├── conftest.py          # Pytest fixtures\n│   ├── test_health.py       # Health check endpoint tests\n│   └── test_config.py       # Configuration tests\n├── requirements.txt         # Python dependencies\n├── pytest.ini               # Pytest configuration\n├── .env.example             # Environment variables template\n└── README.md                # This file\n```\n\n## Setup\n\n1. Create virtual environment:\n```bash\npython -m venv venv\nsource venv/bin/activate  # On Windows: venv\\Scripts\\activate\n```\n\n2. Install dependencies:\n```bash\npip install -r requirements.txt\n```\n\n3. Configure environment:\n```bash\ncp .env.example .env\n# Edit .env with your database credentials and secret key\n```\n\n4. Set up PostgreSQL database:\n```bash\n# Create the database\ncreatedb todo_app\n# Or using Docker:\ndocker run --name postgres-todo -e POSTGRES_DB=todo_app -e POSTGRES_PASSWORD=postgres -p 5432:5432 -d postgres:15\n```\n\n5. Run the application:\n```bash\nuvicorn app.main:app --reload\n```\n\nAPI will be available at http://localhost:8000\n\nAPI documentation: http://localhost:8000/docs\n\n## Environment Variables\n\n| Variable | Description | Default |\n|----------|-------------|---------|\n| DATABASE_URL | PostgreSQL connection string | `postgresql://postgres:postgres@localhost:5432/todo_app` |\n| SECRET_KEY | JWT signing secret (generate with `openssl rand -hex 32`) | - |\n| ALGORITHM | JWT algorithm | `HS256` |\n| ACCESS_TOKEN_EXPIRE_HOURS | Token expiration time | `24` |\n| CORS_ORIGINS | Allowed frontend origins | `["http://localhost:5173"]` |\n| DEBUG | Enable debug mode | `False` |\n\n## API Endpoints\n\n### Health Check\n- `GET /` - Root endpoint, returns API status\n- `GET /health` - Health check with database connection status\n\n## Testing\n\nRun all tests:\n```bash\npytest\n```\n\nRun tests with coverage:\n```bash\npytest --cov=app\n```\n\nRun specific test file:\n```bash\npytest tests/test_health.py\n```\n\n## Development\n\n### Adding a New Router\n1. Create router file in `app/routers/`\n2. Import and include router in `app/main.py`\n\n### Adding a New Model\n1. Create model file in `app/models/`\n2. Import model in `app/models/__init__.py`\n3. Create Alembic migration\n\n### Adding New Schemas\n1. Create schema file in `app/schemas/`\n2. Import schemas where needed\n
